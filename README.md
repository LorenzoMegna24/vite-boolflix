# 🎬 Boolflix

Clone della homepage di Netflix sviluppato come progetto del bootcamp Boolean.

## 🛠 Stack
- Vue 3 + Vite
- JavaScript
- CSS3 / SCSS

## 🎯 Funzionalità
- Layout responsive ispirato a Netflix
- Componenti Vue riutilizzabili
- Struttura dati organizzata per categorie di contenuti
